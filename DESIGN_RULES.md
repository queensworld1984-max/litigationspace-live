# LitigationSpace Design Rules — NEVER BREAK THESE

## Text Colors
- ALL text on dark backgrounds: #ffffff (pure white)
- Secondary text on dark: rgba(255,255,255,0.85) MINIMUM
- Labels/captions on dark: rgba(255,255,255,0.75) MINIMUM  
- NEVER use grey text on dark backgrounds
- Text on white/light backgrounds: #111111

## Backgrounds
- Page background: #0d1117
- Cards: #1a2a3e
- Card inner: #1e3050
- Inputs: #0f172a
- NEVER pitch black backgrounds

## Gold Accent
- Primary gold: #F5A623
- All CTAs, active states, highlights

## Font
- UI: system font
- Analysis output: Poppins

## Rules
- NEVER grey text on dark background
- NEVER build pages that look too dark to read
- ALL analysis text must be white on dark backgrounds
- Placeholder text minimum opacity: 0.5
